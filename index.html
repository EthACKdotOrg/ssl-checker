    Les tests ne se font que sur la connexion aux sites, en aucun cas sur la partie applicative.
    Une mauvaise note ne veut pas dire que les données sont "en accès libre" : tant qu'il y a du SSL/TLS, les données sont chiffrées, et le cassage du chiffrement (décryptage) demandera des efforts (temps machine).
    Si le chiffrement est réputé faible, ça demandera "moins" de temps machine — mais cela reste hors de portée de votre voisin dans tous les cas.
    Les résultats vous inquiètent ? N'hésitez pas à contacter votre banque pour leur signaler que leur site n'offre pas les meilleures garanties de confidentialité.
